# Weather App

A simple weather app built with HTML, CSS, and JavaScript that lets you search any city and instantly view its current temperature and weather conditions. It fetches real-time weather data using a Weather API.

## What it does

- Lets you search for any city
- Shows the current temperature
- Displays weather conditions (like sunny, cloudy, rainy, etc.)
- Shows extra details like humidity, wind speed, or "feels like" temperature
- Clean and simple design that's easy to use

## How to use it

1. Open the app in your browser
2. Type the name of a city in the search box
3. Click the search button (or press Enter)
4. The current weather for that city will appear on the screen

## Built with

- **HTML** – for the structure of the page
- **CSS** – for styling and layout
- **JavaScript** – for handling logic and updating the page
- **Weather API** – for fetching real-time weather data

## How it works

1. You type a city name and search
2. JavaScript sends a request to the Weather API with the city name
3. The API sends back live weather data for that city
4. JavaScript takes that data and updates the page to show it, without needing to reload

## Setup (if using an API key)

1. Sign up on the weather API website and get a free API key
2. Open the JavaScript file
3. Paste your API key where indicated in the code
4. Save the file and open the app in your browser

## Notes

- You need an internet connection since the app fetches live weather data
- Make sure your API key is valid and correctly added, or the weather data won't load
